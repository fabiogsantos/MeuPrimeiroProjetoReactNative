# Instalar React Native
npm install -g react-native-cli 

# Para emulador Android, configuração: 
Usar Next 5
Usar Manshimelow x86 para obter melhor peformance no desktop.
Instalar o driver "Intel x86 Emulator Accelerator (HAXM installer)" para obter melhor peformance no desktop.
Minimum SDK Usar: API 23: Android 6.0 (Marshmallow)

# Para emular o projeto no anroid
$ react-native run-android

# Nota: Aplicação desktop para React-Native "Expo XDE". Para usar linha de comando usar create-react-native-app e no Device instalar o Expo atraves da loja.
$ npm i -g create-react-native-app

# Comando para criar o projeto usando o Expo
$ create-react-native-app <nome do projeto>

# Comando para rodar o projeto usando o Expo
$ npm Start
